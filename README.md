# ENGR-103H-Final
Game I created for my class final

RULES:
Use the left button (D4) to start the game and the right button (D5) to play.
 - After the game starts, a white light will begin moving along the neopixels.
 - The target light will be in color.
 - Press the right button when the moving light is overlapped with the target light to advance rounds.
 - As rounds get faster, it may be necessary to let the moving light pass the target light to time it up better.
 - Every round is +1 to your score.
 - Accumulate the highest score possible.
Flip switch (D7) for HARD MODE.
 - In HARD MODE, the game operates similarly, except you lose if the moving light passes the target light.
